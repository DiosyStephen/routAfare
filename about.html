About routAfare

routAfare is a modern bus fare calculator and prediction system designed to improve the fairness and transparency of private bus fare payments across Sri Lanka. Developed by Team Xterminate from Lyceum International School, the project combines technology, data science, and accessibility to create a smarter transport experience for everyone.

Background

Many passengers face difficulties negotiating bus fares with conductors on private buses. Prices often vary for the same routes, leading to frustration and disputes. Busly addresses this issue by introducing a reliable system that predicts accurate and fair bus fares using data and machine learning.

Project Summary

The project developed a crowdsourced bus fare predictor app using Telegram as the main user interface. It integrates a machine learning algorithm built in Python to calculate estimated fares for any given route. The backend system is powered by the latest Python libraries and frameworks, ensuring accuracy and performance.

Objectives

  • Develop a working calculator and predictor to estimate bus fares.
  • Promote fairness in private bus transactions and reduce fare disputes.
  • Build a user-friendly beta version of the app.
  • Expand the system to be used island-wide.

Methodology

The project began by collecting a dataset of bus fares from the Kurunegala region. Various Machine Learning models were tested — including Linear Regression and Gradient Boosting — to find the most accurate predictor. The team used Flask to integrate the Telegram API with the backend, allowing seamless real-time interactions with users.

Tools & Technologies

  • Python
  • Flask
  • Google Colab
  • Excel (for dataset preparation)
  • National Transport Commission (NTC) data
